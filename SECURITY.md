# Security Policy

## Supported Versions

MirrorMe is currently a prototype and does not maintain long-term support branches.
Security fixes are applied on the main branch.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive reports.

Instead, report privately by contacting the repository owner through GitHub:

- Open the repository Security tab and use private reporting when available.
- Or open a private communication channel via the repository owner profile.

Please include:

- A clear description of the issue
- Reproduction steps
- Impact assessment
- Any proof-of-concept details needed to validate

You can expect an acknowledgement as soon as practical after receipt.
